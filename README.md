# U.S. Infant Mortality Rate Analysis

Analysis of factors affecting infant mortality rates across 50 U.S. states using regression modeling in R.

## Overview
Investigated how political affiliation, U.S. region, and healthcare system 
performance affect infant mortality rates using 2021 CDC data.

## Methods
- Exploratory data analysis and visualization
- Multiple linear regression with stepwise variable selection
- Ridge regression for improved coefficient estimation
- Influential observation diagnostics and assumption checking

## Results
Final model achieved Adjusted R² of 0.645, identifying healthcare system 
performance, low birth weight rate, and twin birth rate as significant predictors.

## Tools & Technologies
- R (ggplot2, olsrr, glmnet)

## Data Sources
- National Vital Statistics Reports (CDC, 2021)
- Pew Research Center State Political Affiliation Data
- Commonwealth Fund 2022 State Health System Scorecard
